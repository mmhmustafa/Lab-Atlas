# Format: "<src-node> <dst-node>" (one direction per line).
#
# Why these are unreachable:
#   isp1 <-> isp2
#     Atlas is dual-homed to two independent upstream providers. There is
#     no link and no route between the ISPs themselves - Atlas does not
#     provide transit between them. See docs/routing.md.
#   isp1/isp2 <-> branch1-4
#     Branch site loopbacks/LANs (10.255.1.0/24, 172.16.100.0/22) are
#     deliberately excluded from RM-TO-ISP (the edge1/edge2 route-map
#     that filters what gets advertised to the ISPs), so branch sites
#     stay private WAN-only and are never leaked to the internet. See
#     inventory/devices.yaml and docs/routing.md.
